# Robo-Friend React.js App

> React app to search Robo Friends

## Quick Start

```bash
# clone the project
git clone https://github.com/samhayder/robofriend-react.git

# Install dependencies
npm install

# Serve on localhost:3000
npm start

# Build for production
npm run build
```
